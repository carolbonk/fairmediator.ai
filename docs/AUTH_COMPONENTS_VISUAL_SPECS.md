## Design Notes

### Shadow Depth Hierarchy (from subtle to prominent)

1. **Inset** - Inputs, pressed buttons, recessed areas
2. **Flat** - Minimal elevation, subtle depth
3. **Raised (neu)** - Standard cards, default buttons
4. **Elevated (neu-lg)** - Hover states, prominent cards
5. **Floating (neu-xl)** - Modals, overlays, dropdowns

### Color Psychology in Auth

- **Blue gradients (primary)**: Trust, security, professionalism
- **Green states (success)**: Confirmation, safety, achievement
- **Red states (error)**: Urgency, warning, correction needed
- **Yellow states (warning)**: Caution, attention, opportunity
- **Gray tones (neutral)**: Balance, calm, sophistication
