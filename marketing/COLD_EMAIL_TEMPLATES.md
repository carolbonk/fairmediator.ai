## Personalization Checklist

Before sending, customize each email with:

- [ ] {{FirstName}} - Use their actual first name
- [ ] {{CompanyName}} - Their law firm or company
- [ ] {{MutualConnection}} - If applicable, mention shared connection
- [ ] {{SpecificDetail}} - Reference their practice area, recent case, LinkedIn post, etc.
- [ ] {{YourName}} - Your actual name (builds trust vs generic "FairMediator Team")

**Example Personalization:**

```
Generic:
"Hi, I built a tool for conflict checking."

Personalized:
"Hi Sarah, I saw your LinkedIn post about the challenges of mediator selection in IP disputes. I built FairMediator to solve exactly that problem."
```

**Response Rate:**
- Generic email: 5-10%
- Personalized email: 20-35%

## Sending Best Practices

### Timing
- **Best days:** Tuesday, Wednesday, Thursday
- **Best times:** 6-7am, 12-1pm, 4-5pm (when they check email)
- **Worst times:** Monday morning (inbox overload), Friday afternoon (checked out)

### Volume
- **Week 1:** 10-20 emails/day (test subject lines, track open rates)
- **Week 2:** Scale to 50-100 emails/day (use templates with highest response rate)
- **Week 3+:** Automate with tools (Lemlist, Mailshake, Outreach.io)

### Tracking
Use email tracking tools to monitor:
- Open rate (target: 40-60%)
- Click rate (target: 10-20%)
- Response rate (target: 15-30%)
- Demo booking rate (target: 5-10% of emails sent)

**Tools:**
- HubSpot Sales (free)
- Mixmax (Gmail)
- Mailtrack (simple open tracking)

## A/B Testing Framework

### Variables to Test
1. **Subject line:** Question vs statement vs data vs personalization
2. **Email length:** Short (3 sentences) vs medium (5-7 sentences) vs long (case study)
3. **CTA:** "Book demo" vs "Reply to this email" vs "Try free"
4. **Tone:** Formal vs casual vs data-driven
5. **Hook:** Problem vs solution vs story vs data

### Example A/B Test

**Version A (Short, Direct):**
```
Subject: 40% of mediators have conflicts. Did you check yours?

{{FirstName}},

FairMediator finds conflicts mediators don't disclose.

Try 3 free checks: [link]

[Your Name]
```

**Version B (Story-Based):**
```
Subject: How we found a $15K conflict in 30 seconds

{{FirstName}},

[Full Template #1 from above]
```

**Test with:** 50 emails each, same audience, same day/time

**Measure:** Open rate, click rate, response rate, demo bookings

**Winner:** Version with highest demo booking rate (not just opens)
