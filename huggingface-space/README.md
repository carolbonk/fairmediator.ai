# FairMediator AI Demo

**Intelligent Mediator Matching & Screening Platform**

## Features

- **Full Mediator Analysis** - Complete screening with ideology, conflicts, and entities
- **Ideology Classification** - Detect political leanings (liberal/conservative/neutral)
- **Conflict Detection** - Identify potential conflicts of interest
- **Entity Extraction** - Extract organizations, people, and locations

## Technology

- HuggingFace Transformers (100% FREE)
- DistilBERT for sentiment analysis
- BART for zero-shot classification
- BERT for Named Entity Recognition

## Use Cases

- Screen mediators for potential bias
- Check for conflicts of interest before selection
- Analyze mediator backgrounds and affiliations
- Extract key information from profiles
